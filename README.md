# IPL-dashboard
LiveDemo:
https://gunaipl.ccbp.tech/

Description 
Developed a dynamic, responsive sports dashboard Single Page Application (SPA) utilizing ReactJS to aggregate and display cricket tournament data. Integrated asynchronous API fetching routines to consume external REST endpoints, dynamically parsing JSON payloads to render team rosters, individual match logs, and win/loss statistics. 

Features
-Single page application
-API fetching to fetch the details about IPL team selected
